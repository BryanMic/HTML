Flexbox VS Grid

Quelles sont les principales différences?

Comme montré ci-dessus, flexbox n’agit que sur une dimension, alors que grid permet d’agir sur les deux en même temps.

Flexbox agit sur les enfants, ne peut agir que sur un axe et repousse les elements.

Grid permet d’agir sur les deux axes en même temps, agit quant à lui sur les parents. Il est plus facile de manipuler des objets sur une page avec grid plutot que flexbox
